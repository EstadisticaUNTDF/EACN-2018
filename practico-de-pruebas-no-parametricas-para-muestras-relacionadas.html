Capítulo 16 Práctico de Pruebas no paramétricas para muestras relacionadas

Es posible realizar el equivalente no paramétrico de un anova simple con bloques. Hay al menos dos formas de hacerlo en R. Una es con la función friedman.test() del paquete stats. Tiene una interfaz de vectores y otra de formula. La de vectores tiene tres argumentos, y la variable de respuesta, groups la variable tratamiento, y blocks los bloques. La interfaz de formula usa la ya conocida forma y ~ groups pero para indicar los bloques usa una barra recta de esta manera y ~ groups | blocks.

alpha <- 0.05
c <- nrow(out$means)
Z <- abs(qnorm(alpha/(c*(c-1))))
r <- unique(out$means$r)
VC <- Z * sqrt(r*c*(c+1)/6)
